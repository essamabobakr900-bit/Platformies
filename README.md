# My Godot Game

## Description

This is a 2D platformer game made using Godot 4.

## Features

* Player movement
* Jump system
* Collision detection
* Overcoming enemies

## How to Play

Use arrow keys to move and space to jump.
don't touch enemies or the game will be restarted.
collect your coins as much as possible in order to get the highest score.
Your target is to reach the win flag, so try and find a way to reach it.

## Tools Used

* Godot Engine
* GDScript
